Javascript Basics

Javascript is a language that is supported by all browsers and its mostnly used for enchancing webpages making it more interactive and livley. This is an essential language to learn as the majority of the IT industry are looking for people with knowledge of javascript allowing you to get a job more easily. Javascript will teach you how to use variables, arrays, loops, conditional statements and some other advanced javascript concepts. Javascript is perfect for you if you want to learn about web application development allowing you to build on your html and css skills and even if you can't grasp the syntax of javascript you can use modern tools like Coffee script, Typescript, and Dart which allows you to code in a different language and then auto-convert into javascript.
